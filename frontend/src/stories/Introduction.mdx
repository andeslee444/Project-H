# Mental Health Practice Management System

Welcome to the component library for the Mental Health Practice Management System. This Storybook showcases all the UI components and patterns used throughout the application.

## 🏥 About the System

This is a comprehensive scheduling and waitlist management system designed specifically for mental health practices. The system features:

- **Patient Management**: Track patient information, appointments, and treatment history
- **Provider Scheduling**: Advanced scheduling with availability management
- **Intelligent Matching**: AI-powered patient-provider matching based on specialties and needs
- **Waitlist Management**: Automated waitlist with smart notifications
- **HIPAA Compliance**: Built with security and privacy at its core
- **Performance Optimized**: Fast, responsive interfaces for all users

## 🎨 Design System

Our design system is built on these core principles:

### Colors
- **Primary**: Blue shades for primary actions and navigation
- **Success**: Green for positive actions and confirmations
- **Warning**: Amber for cautions and alerts
- **Danger**: Red for destructive actions and errors
- **Neutral**: Gray scale for text and backgrounds

### Typography
- **Font**: System font stack for optimal performance
- **Sizes**: Consistent scale from xs to xl
- **Weights**: Regular (400), Medium (500), Semibold (600), Bold (700)

### Spacing
- Based on a 4px grid system
- Common values: 4, 8, 12, 16, 20, 24, 32, 48, 64px

### Components
All components are:
- ✅ Fully accessible (WCAG 2.1 AA compliant)
- ✅ TypeScript typed
- ✅ Tested with >80% coverage
- ✅ Responsive and mobile-friendly
- ✅ Dark mode compatible
